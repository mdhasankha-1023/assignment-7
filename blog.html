1. What is difference between map, filter, ForEach, and find methods ?

Ans: Map, Foreach, Filter and Find are all JavaScript methods. Some differences exist between them. For example,
Map: If map method is invoked on an array, the map method takes all the objects from this array ito a new array and returns it.
`ForEach: forEach method fetches all objects from an array but does not assign a new array and returns nothing
Filter: When the filter method is invoked on an array, filter method checks the objects in the array against the condition, returning only those objects that match the condition.
Find: filter method and find method work the same way. Just find method returns the first object in the array that match the condition

2. What is difference between local storage and session storage ?

Ans: Local storage and session storage are the same thing. They store browsing data. the difference between them is that local storage store browsing data permanently, on the other hand session storage store browsing data temporarily, which is deleted when the browser reloads.

3. What is event loop and how it works ?

Ans: The Event Loop has one simple job to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.
